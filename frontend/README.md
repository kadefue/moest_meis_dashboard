# M&E Dashboard - React Frontend

This folder is a Vite + React starter for the M&E Dashboard frontend.

To run locally:

```bash
cd /Users/kadefue/MoEST_M_n_E/frontend
npm install
npm run dev
```

By default the app expects the backend API at `http://127.0.0.1:8000/api`. To change, set `VITE_API_BASE` in your environment.
